@plnv/canvas — демо

Пример библиотеки @plnv/canvas: несколько директив-слоёв рисуют в один родительский canvas — очередь отрисовки и размеры задаёт хост plnvCanvas2d.

Снизу вверх по стеку отрисовки: фон блока (CSS), горы MountainLayer (можно скрыть), затем пунктирные орбиты и диски SunMoonLayer (светлая тема — солнце по времени суток; тёмная — луна, фаза от числа месяца, 13-е — полнолуние), сверху — снег SnowLayer (ведите мышью). «Перерисовать» — новый силуэт гор; тема на html (data-theme): по умолчанию как в ОС; кнопка справа циклически: системная → светлая → тёмная → системная.
